1. What is C++? Introduction and Features  
C++ is a programming language used to create software, games, and 
applications. It was developed by Bjarne Stroustrup.  
It is an improved version of C language.  
Features of C++   
• simple and fast   
• supports Object-Oriented Programming (OOP)  
• reusable code (no need to write again)   
• secure (data can be hidden)   
• works on many systems  
2. What is Encapsulation in C++? Explain  
Encapsulation means keeping data safe inside a class and using functions to 
access it.  Data is hidden using private  
3. What is Inheritance in C++?  
Inheritance means one class uses properties of another class.  
It helps to reuse code.  
4. What is Polymorphism in C++?  
Polymorphism means one name, many forms.  
A function can behave differently based on context.

ATM Management System (C++)  
1. Introduction 
The Quiz Game System is a console-based application designed to evaluate a 
user's knowledge through a series of multiple-choice questions. It provides an 
interactive interface where users can register, answer questions, and view their final 
scores. The system utilizes C++ features to ensure data integrity and a smooth user 
experience. 
2. Objectives of the Project 
● To create an interactive platform for learning and self-assessment. 
● To implement Encapsulation by keeping user scores and question data 
secure. 
● To use File Handling to store high scores or a question bank externally. 
● To demonstrate the use of Control Flow (if-else, loops) and Data 
Structures (classes, arrays). 
3. Methodology / System Design 
The system is designed using a Modular Approach. Each component is handled 
by a specific part of the class: 
● User Management: Handles user names and score tracking. 
● Question Bank: A collection of objects containing the question text, options, 
and the correct answer. 
● Logic Engine: Compares user input with the correct answer and updates 
the score.

• Language: C++

Input/Output
